# 📚 Cyberia Practice: Сервис управления каталогом книг

Проект представляет собой API-ориентированное Laravel-приложение для каталогизации книг, управления избранным, отзывами, рейтингами с полноценной административной панелью (Filament) и системой логирования действий.

---

## 🛠️ Технологический стек
*   **Backend**: PHP 8.4 (Laravel 12.x)
*   **Database**: PostgreSQL 17
*   **Admin Panel**: Filament PHP 3.x
*   **API Docs**: Dedoc Scramble
*   **Containerization**: Docker & Docker Compose
*   **Media**: Spatie Media Library (загрузка обложек книг и аватаров авторов)

---

## 📋 Отчет о проделанной работе

В рамках практики были успешно реализованы следующие задачи:

### ⚙️ Базовая настройка и авторизация
*   **Задача 0**: Подготовлено локальное окружение (IDE, Bruno/Postman, Docker).
*   **Задача 1**: Развёрнута Docker-архитектура проекта: контейнеры `php-fpm`, `nginx`, `pgsql` (PostgreSQL 17).
*   **Задача 2**: Спроектирована база данных, созданы модели и миграции для сущностей `User`, `Author`, `Book`, `Genre`, `Review`. Настроены сидеры и фабрики для генерации реалистичных демонстрационных данных.
*   **Задача 3**: Реализовано беспарольное API-авторизация (вход/регистрация) по коду из СМС/email (код пишется в логи Laravel). Поддерживает расширяемость через сервисную абстракцию.
*   **Задача 4**: Созданы методы для просмотра и обновления профиля авторизованного пользователя.

### 📖 Каталог книг и отзывы
*   **Задача 5**: Реализованы API-методы просмотра каталога книг, авторов (с выводом количества книг и последней книги) и жанров с пагинацией.
*   **Задача 6**: Созданы изолированные классы фильтрации (`BookFilter`) и сортировки (`BookSorter`) для книг.
*   **Задача 7**: Добавлен функционал «Избранное» для пользователей (добавление, удаление, просмотр с пагинацией и очистка списка).
*   **Задача 8**: Интегрирован пакет `Spatie Media Library` для работы с медиа: загрузка обложек для книг и аватаров для авторов.
*   **Задача 9**: Добавлены отзывы к книгам (оценка, текст, плюсы/минусы, анонимность) со статистикой рейтинга книг.
*   **Задача 10**: Реализован рейтинг отзывов (лайки/дизлайки) с возможностью повышения/понижения рейтинга отзыва авторизованными пользователями.
*   **Задача 11**: Полностью локализованы все системные сообщения, ошибки валидации и атрибуты (`lang/ru` и `lang/en`).

### 👑 Панель администратора
*   **Задача 20**: Настроены CRUD-ресурсы в Filament для управления книгами и авторами, добавлены менеджеры связей (RelationManagers), отображение отзывов и управление медиафайлами.
*   **Задача 21**: Реализовано автоматическое логирование действий с сущностями (регистрация, вход, создание/редактирование/удаление книг, авторов и жанров) на базе Eloquent Observers. Настроена read-only Filament-панель для просмотра системных логов.

### 🔍 Расширенная фильтрация и документация
*   **Задача 23**: Каталог книг расширен новыми характеристиками (издательство, страницы, размер, тип обложки, вес, возрастное ограничение). Реализован эндпоинт GET `/api/books/filters` для динамического пересчёта доступных фильтров.
*   **Задача 24**: Расширена сортировка книг (по году издания, по количеству комментариев).
*   **Задача 25**: Настроено автоматическое генерирование API-документации с помощью пакета `Scramble`.

## 🚀 Быстрый старт

### 1. Подготовка окружения
Скопируйте настройки окружения:
```bash
cp .env.example .env
```

### 2. Запуск контейнеров
Соберите и запустите Docker-контейнеры:
```bash
docker compose up -d --build
```

### 3. Накатывание миграций и сидов
Выполните миграции и наполните базу реальными демонстрационными данными:
```bash
docker exec -it cyberia_practice-php-fpm-1 php artisan migrate --seed
```
